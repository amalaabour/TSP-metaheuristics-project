
# TSP Metaheuristics Project

This project presents a comparative analysis between:

- Nearest Neighbor heuristic (NN)
- Tabu Search metaheuristic

for solving the Traveling Salesman Problem (TSP).

## Project Structure

- **small_instances / medium_instances / large_instances**: TSPLIB instances  
- **Notebooks**: implementation and experiments  
- **results.csv**: experimental results  
- **Compte_rendu.pdf**: project report  
  *(To access the clickable links in the document, it is recommended to download the PDF.)*
